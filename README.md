## 📖 About The Project
**PlotMyPath** is a modern, responsive web application designed to make travel planning effortless. Built with React and powered by AI, it helps travelers create day-by-day itineraries, track budgets, and manage packing lists—all in one sleek interface.

### ✨ Key Features
* **AI-Powered Itinerary:** Generate custom trip plans based on destination and duration.
* **Smart Budget Handler:** Track expenses, add notes, and manage group splits.
* **Packing Checklist:** Never forget your essentials with interactive progress tracking.
* **Dark/Light Mode:** Premium UI/UX with seamless theme switching.
* **PWA Ready:** Install it on your phone like a native app.
